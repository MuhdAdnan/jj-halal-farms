A custom Django-based admin dashboard for managing products, customers, orders, and payments for an agricultural e-commerce platform.  
This project was built with real production use in mind, clean architecture, modular apps, and secure payment handling.

Key Features
- Public product listing and search UI
- Customer registration with email verification
- Customer cart, checkout, order history, and inbox
- Paystack payment (with webhook verification)
- Pay‑on‑delivery for farm pickup
- Admin dashboard with products, orders, customers, and profile management
- Admin messaging to customers (inbox)

Tech Stack
- Django 6.0.1
- PostgreSQL
- Bootstrap 5
- Paystack
